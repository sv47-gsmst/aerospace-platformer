# 🚀 Orbital Cadet

Welcome to **Orbital Cadet**! This is a slick, vanilla JavaScript space platformer built completely from scratch with zero dependencies. No frameworks, no build tools, no bloated node modules—just pure HTML5 Canvas magic. 

Whether you want to speedrun through the hand-crafted levels or see how long you can survive in the procedural Infinite Mode, this cadet is ready for launch.

## ✨ Features
* **Two Game Modes:** Tackle progressive levels or survive the endless, procedurally generated Infinite Mode.
* **Silky Smooth Physics:** A custom delta-time (`dt`) engine ensures the game runs at the exact same speed on a standard 60Hz laptop as it does on a 144Hz gaming rig.
* **"The Juice":** Packed with modern platformer game-feel mechanics like Coyote Time, Jump Buffering, particle exhaust trails, and screen shake. 
* **Retro Audio:** Custom synthesized sound effects generated natively in your browser using the Web Audio API. 
* **Auto-Saving:** Uses LocalStorage to quietly save your Infinite Mode high score and level unlocks so you never lose your progress.
* **Play Anywhere:** A fully responsive canvas that scales to any screen, complete with automatic on-screen touch controls for mobile devices.

## 🎮 How to Play
* **Move:** `A` / `D` or `Left Arrow` / `Right Arrow`
* **Jump:** `W` / `Up Arrow` / `Space` (Double jump supported!)
* **Pause / Menu:** `ESC`
* **Mobile:** Just tap the on-screen UI buttons!

## 🛠️ Installation & Setup
Literally nothing. No `npm install`, no servers to spin up. 
1. Clone or download this repository.
2. Double-click `index.html` to open it in your web browser.
3. Start jumping.
